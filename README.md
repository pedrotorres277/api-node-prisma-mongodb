# 🚀 API CRUD - Node + Prisma + MongoDB

Projeto simples de API REST utilizando:

- Node.js
- Express
- Prisma ORM
- MongoDB Atlas
- HTML, CSS, JS puro

## 📌 Funcionalidades

- Criar usuário
- Listar usuários
- Atualizar usuário
- Deletar usuário

## ▶️ Como rodar

1. Clone o projeto
2. Instale as dependências:

npm install


3. Crie um `.env` com:

DATABASE_URL="SUA_URL_DO_MONGODB"


4. Rode:

node server.js
